# Moderated Session Runbook

## Pre-Session (15 Minutes)

- Confirm participant consent status.
- Prepare repository and environment freeze checklist from `baseline-rubric.md`.
- Open session script and findings template.
- Start recording (if consented).

## Live Session Flow

1. Introduction and consent
2. Baseline questions
3. Task scenarios (no hints unless participant is blocked)
4. Debrief questions
5. Confidence and satisfaction capture

## Facilitator Rules

- Do not teach during task execution unless safety or total blockage requires intervention.
- Use neutral prompts: "What are you expecting to happen here?"
- Log exact wording for participant confusion statements.
- Mark each intervention with timestamp and reason.

## Post-Session (10 Minutes)

- Complete findings log before memory decay.
- Classify each issue by severity.
- Add candidate backlog entries with validation tests.
- Link captured evidence artifacts to the daily summary.

## Daily Wrap-up

- Aggregate completion, time, and error metrics.
- Re-score backlog priorities based on observed frequency.
- Select next-day top fixes (Critical first, then highest score).
