# Participant Screener (Operator and Agent Integrator)

## Usage

Use this screener before scheduling moderated sessions. Target balanced representation by persona and experience.

## Mandatory Questions

1. Which profile best matches you?
- Operator (uses tools to understand codebases)
- Agent Integrator (integrates APIs into agents or automation)

2. How would you rate your API experience?
- Novice
- Intermediate
- Advanced

3. How often do you use Swagger/OpenAPI docs?
- Never
- Occasionally
- Weekly
- Daily

4. Have you used code intelligence or RAG tools before?
- No
- Yes (briefly)
- Yes (regularly)

5. Do you use accessibility features while working?
- No
- Yes: screen reader
- Yes: zoom/magnification
- Yes: keyboard-only navigation
- Yes: other

## Selection Rules

- Aim for 6 Operator and 6 Agent Integrator participants.
- Include at least 2 participants who rely on assistive features.
- Balance novice/intermediate/advanced distribution within each persona.

## Exclusion Criteria

- Direct maintainers of this repository for external sessions.
- Participants unable to complete consent requirements.
